- Domain: Independent and Self-help

- Task: Meal Preparation

- Sub task: Making a sandwich

- Description: Teaching the client to independently prepare a simple sandwich as part of developing basic cooking skills.

- Materials:
  * Bread
  * Sandwich fillings (e.g., cheese, ham, lettuce)
  * Butter or spread (optional)
  * Knife (appropriate for client's safety level)
  * Plate
  * Visual task analysis chart
  * Timer (optional)

- Procedure:
  1. Set up materials in the kitchen
  2. Present visual task analysis chart
  3. Provide verbal instruction: "Let's make a sandwich"
  4. Use least-to-most prompting hierarchy:
     a. Verbal prompt
     b. Gestural prompt
     c. Partial physical prompt
     d. Full physical guidance
  5. Gradually fade prompts as client demonstrates increased independence
  6. Provide immediate reinforcement for successful completion

- Data Collection:
  * Record level of prompting required for each step
  * Track successful independent completions
  * Note duration of task completion (if using timer)

- Variations:
  * Introduce different types of sandwiches
  * Incorporate spreading butter or condiments
  * Practice cutting sandwich in half (if appropriate)

- Reinforcement:
  * Verbal praise for attempts and successful completion
  * Token system with preferred reward for consistent performance
  * Natural reinforcement of eating the prepared sandwich

- Skills Developed:
  * Fine motor skills
  * Following multi-step instructions
  * Task sequencing
  * Food preparation skills
  * Safety awareness in kitchen
  * Independence in daily living skills
