- Domain: Sensory Motor Skill

- Task: Fine Motor Precision

- Sub task: Bead Threading

- Description: The client will practice threading beads onto a string to improve fine motor skills, hand-eye coordination, and visual perception.

- Materials:
  - Large wooden beads with holes
  - Thick string or shoelace
  - Timer
  - Data collection sheet

- Procedure:
  1. Present the beads and string to the client.
  2. Demonstrate how to thread one bead onto the string.
  3. Provide hand-over-hand guidance if needed for the first few beads.
  4. Gradually reduce physical prompts as the client becomes more proficient.
  5. Set a timer for 2 minutes and encourage the client to thread as many beads as possible.
  6. Count and record the number of beads threaded.

- Data Collection:
  Record the number of beads threaded independently within the 2-minute time frame for each session.

- Variations:
  - Use beads of different sizes, shapes, or colors to increase difficulty or add a sorting component.
  - Increase or decrease the time limit based on the client's skill level.
  - Create patterns for the client to follow while threading.

- Reinforcement:
  Provide verbal praise for each bead threaded and offer a preferred item or activity upon completion of the task.

- Skills Developed:
  - Fine motor dexterity
  - Hand-eye coordination
  - Visual perception
  - Task persistence
  - Following instructions
  - Concentration
